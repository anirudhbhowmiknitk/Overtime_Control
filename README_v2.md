# Overtime Control v2

Use `unified_platform_v2.py` for the complete one-platform experience, or run the individual `worker_portal_v2.py`, `engineer_portal_v2.py`, and `supervisor_portal_v2.py` files separately.

Quick start:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
streamlit run unified_platform_v2.py --server.port 8501
```
